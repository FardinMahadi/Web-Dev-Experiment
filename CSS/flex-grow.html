Example 1: Equal Distribution

Item 1
flex-grow: 1
Item 2
flex-grow: 1
Item 3
flex-grow: 1
.container { display: flex; } .item { flex-grow: 1; /* All items grow equally */ }

How This Works

When all items have the same flex-grow value (1 in this case), the available space in the container is distributed equally among them.

Each item receives the same amount of extra space beyond its initial content size.

Example 2: Unequal Distribution

Item 1
flex-grow: 1
Item 2
flex-grow: 2
Item 3
flex-grow: 3
.container { display: flex; } .item:nth-child(1) { flex-grow: 1; } /* 1 part */ .item:nth-child(2) { flex-grow: 2; } /* 2 parts */ .item:nth-child(3) { flex-grow: 3; } /* 3 parts */

How This Works

When items have different flex-grow values, the available space is distributed proportionally.

In this case, the total growth factor is 1 + 2 + 3 = 6. Item 1 gets 1/6 of the space, Item 2 gets 2/6, and Item 3 gets 3/6.

The items grow in a 1:2:3 ratio.

Example 3: Mixed Growth

Fixed Size
flex-grow: 0
width: 120px
Growing Item
flex-grow: 1
Growing More
flex-grow: 2
.container { display: flex; } .item:nth-child(1) { flex-grow: 0; /* Won't grow */ width: 120px; /* Fixed width */ } .item:nth-child(2) { flex-grow: 1; } /* Takes 1 part of remaining space */ .item:nth-child(3) { flex-grow: 2; } /* Takes 2 parts of remaining space */

How This Works

Item 1 has flex-grow: 0 and a fixed width, so it won't grow.

The remaining space is distributed between Item 2 and Item 3 in a 1:2 ratio.

Item 2 gets 1/3 of the remaining space, and Item 3 gets 2/3 of the remaining space.
